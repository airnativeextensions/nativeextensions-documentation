Property Detail
audioOutputproperty
public var audioOutput:String = default

Sets the output to either default, speaker or receiver. Note: This is potentially incompatible with background audio. On iOS this affects all application audio, not just the audio from this player.

backgroundAudioEnabledproperty 
public var backgroundAudioEnabled:Boolean = false

Enables background audio mode for this player.

You should only have one player that utilises background audio, using more than one player will have undefined results.

playbackSpeedEnabledproperty 
public var playbackSpeedEnabled:Boolean = false

If you wish to control the playback speed of the audio in this player you should enable this option. This will prepare the player with the appropriate resources to change the playback speed.

The default value is false.
