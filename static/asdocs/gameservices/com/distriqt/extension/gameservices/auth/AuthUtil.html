Interfacepublic interface AuthUtil extends flash.events.IEventDispatcher

AuthUtil provides static utility methods to,

  • Acquire ClientLogin, OAuth2, ID, or offline access tokens for Google accounts.
  • Invalidate ClientLogin or OAuth2 tokens.

getToken()method 
public function getToken(scope:String):Boolean

Gets a token or authorisation to be consumed by some other services on behalf of a specified user account.

How the token can be consumed depends on the scope string provided.

Note that this method requires substantial network IO and is run asynchronously off the UI thread

For Google Play Services this will return the current players Google Authentication Token.

For GameCenter this returns the Identity Verification Signature from the generateIdentityVerificationSignatureWithCompletionHandler function. more information.

Parameters

scope:String — (optional) A string representing the authentication scope. To specify multiple scopes, separate them with a space (for example, "oauth2:scope1 scope2 scope3").

Returns
Boolean — true if the operation was started successfully and false if it's not supported
