addImage()method 
public function addImage(data:BitmapData, imageOptions:ImageOptions = null):Boolean

Adds a raster image print command to the command buffer A specified area of BitmapData is binarized according to the mode, halftone, and brightness parameters and converted into a raster image. The converted image is compressed or not compressed before transmission according to the compress parameter value. One pixel of an image corresponds to one dot of the printer. When a transparent color is contained in the image, the background of the image is assumed to be white.

Parameters

data:BitmapData
 
imageOptions:ImageOptions (default = null)

Returns
Boolean — true if successfully added to the command buffer, false otherwise
addPulse()method 
public function addPulse(drawer:int, time:int):Boolean

Adds a drawer kick command to the command buffer. Sets the drawer kick. drawer should be one of:

  • EposFlags.DRAWER_2PIN: Drawer kick connector pin No.2
  • EposFlags.DRAWER_5PIN: Drawer kick connector pin No.5
  • EposFlags.PARAM_DEFAULT (default): Specifies the default (drawer kick connector pin No.2).
time should be one of:
  • EposFlags.PULSE_100: 100-msec signal
  • EposFlags.PULSE_200: 200-msec signal
  • EposFlags.PULSE_300: 300-msec signal
  • EposFlags.PULSE_400: 400-msec signal
  • EposFlags.PULSE_500: 500-msec signal
  • EposFlags.PARAM_DEFAULT (default): Specifies the default (100-msec signal).

Parameters

drawer:int (default = NaN) — Specifies the drawer kick connector
 
time:int (default = NaN) — Specifies the on time of the drawer kick signal

Returns
Boolean — true if sent successfully and false otherwise

forcePulse()method 
public function forcePulse(drawer:int, time:int, timeout:int):Boolean

Forcibly sends the drawer kick command. drawer should be one of:

  • EposFlags.DRAWER_2PIN: Drawer kick connector pin No.2
  • EposFlags.DRAWER_5PIN: Drawer kick connector pin No.5
  • EposFlags.PARAM_DEFAULT (default): Specifies the default (drawer kick connector pin No.2).
time should be one of:
  • EposFlags.PULSE_100: 100-msec signal
  • EposFlags.PULSE_200: 200-msec signal
  • EposFlags.PULSE_300: 300-msec signal
  • EposFlags.PULSE_400: 400-msec signal
  • EposFlags.PULSE_500: 500-msec signal
  • EposFlags.PARAM_DEFAULT (default): Specifies the default (100-msec signal).

Parameters

drawer:int (default = NaN) — Specifies the drawer kick connector
 
time:int (default = NaN) — Specifies the on time of the drawer kick signal
 
timeout:int (default = NaN) — Specifies the timeout period before completion of command transmission in milliseconds (0 to 600000, or EposFlags.PARAM_DEFAULT is 10000)

Returns
Boolean — true if sent successfully and false otherwise
