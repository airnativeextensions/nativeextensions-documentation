Property Detail
autoOrientproperty
public var autoOrient:Boolean = true

If true then the load process will correct the orientation of the bitmap data loaded using any orientation information contained in the EXIF data of the image.

This is currently unimplemented but is in development testing.

heightproperty 
public var height:int = 0

The height value to use in any resizing. You can use a pixel size or one of the calculated values:

  • THUMBNAIL_SIZE
  • DISPLAY_SIZE

resizeMethodproperty 
public var resizeMethod:String = resizeNone

This is the method that will be taken to resize the image (if required).

Note: The autoOrient process will be performed before any resizing processes.

widthproperty 
public var width:int = 0

The width value to use in any resizing. You can use a pixel size or one of the calculated values:

  • THUMBNAIL_SIZE
  • DISPLAY_SIZE

FULL_RESOLUTIONConstant

This representation contains the biggest, best representation available

RESIZE_BEST_FITConstant 
public static const RESIZE_BEST_FIT:String = resizeBestFit

This method will best fit the image to inside the specified width and height dimensions. The result will be smaller than or equal to the specified dimensions. No cropping will occur.

RESIZE_FILLConstant 
public static const RESIZE_FILL:String = resizeFill

This method fills the specified size, scaling content to fill the entire area and cropping any content falling outside. Use this method if you must have an exact pixel size for your image display, eg thumbnail images or icons.

RESIZE_FILL_NO_CROPConstant 
public static const RESIZE_FILL_NO_CROP:String = resizeFillNoCrop

This method takes the smallest size of the image (either width or height) and makes it equal to the specified width or height value.

This is very similar to the RESIZE_BEST_FIT method except that rather than fitting inside the width and height, the result is guaranteed to fill the width and height specified.

This is useful if you require a specific minimum resolution but don't want to crop the image.

RESIZE_NONEConstant 
public static const RESIZE_NONE:String = resizeNone

The image will not be resized and the original image data returned.

THUMBNAILConstant 
public static const THUMBNAIL:AssetRequest

Returns a thumbnail representation of the asset. The size of the thumbnail is the appropriate size for the platform, and in the correct orientation.
