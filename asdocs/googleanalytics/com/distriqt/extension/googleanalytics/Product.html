Packagecom.distriqt.extension.googleanalytics
Classpublic class Product
InheritanceProduct Object

Product related information for a Google Analytics hit.

Use this class to report information about products sold by merchants or impressions of products seen by users. Instances of this class can be associated with both ProductActions via addProduct(Product) and Product Impressions via addImpression(Product, String).

See also

com.distriqt.extension.googleanalytics.builders.HitBuilder.addProduct()
com.distriqt.extension.googleanalytics.builders.HitBuilder.addImpression()
