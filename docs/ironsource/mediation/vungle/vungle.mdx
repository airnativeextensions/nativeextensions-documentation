1. Create an account with Vungle. You can do so [here](https://v.vungle.com/dashboard/login#newUser).

2. Once your account has been verified you can log in at their partner login [here](https://v.vungle.com/dashboard/login).

## Step 5: Vungle Additional Settings

### COPPA User status

ironSource's mediation lets publishers communicate Vungle COPPA settings directly to the Vungle network. This feature is enabled using ironSource SDK 7.1.12+, Vungle Android Adapter 4.3.15+. Read more about Vungle COPPA API [here](https://support.vungle.com/hc/en-us/articles/360047780372#coppa-api-0-8).

Use the following syntax, to update Vungle's COPPA User Status with the parameter “True”:

```actionscript
IronSource.service.setMetaData( "Vungle_coppa", "true" );
```

Use the following syntax, to update Vungle's COPPA User Status with the parameter “False”:

```actionscript
IronSource.service.setMetaData( "Vungle_coppa", "false" );
```
