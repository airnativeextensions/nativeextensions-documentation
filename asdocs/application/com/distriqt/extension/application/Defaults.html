Interfacepublic interface Defaults extends flash.events.IEventDispatcher

The defaults allow you to setup some information for the user, to allow an application to customize its behavior to match a user's preferences. For example, units of measurement, or refresh intervals.

Under iOS this uses the NSUserDefaults API and under Android it uses the SharedPreferences API.

  • https://developer.apple.com/library/ios/documentation/cocoa/reference/foundation/Classes/NSUserDefaults_Class/Reference/Reference.html
  • http://developer.android.com/reference/android/content/SharedPreferences.html

Property Detail
useSharedDefaultsproperty
useSharedDefaults:Boolean

Use the shared defaults to save values. The default is false, which will save values into a custom object to avoid conflict / clashes with other values stored within your applications other native components. If you wish to access the shared defaults / preferences, eg to access the GDPR data transparency values, you can set this to true. This will then change the access to the applications shared values.
