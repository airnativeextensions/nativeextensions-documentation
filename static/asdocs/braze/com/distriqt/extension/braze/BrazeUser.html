addAlias()method
public function addAlias(alias:String, label:String):Boolean

Adds an alias for the current user. (alias, label) pairs can exist on one and only one user. If a different user already has this alias or external user id, the alias attempt will be rejected on the server (even if this method returns true).

Parameters

alias:String — An alias for the current user
 
label:String — The label of the alias. Used to differentiate it from other aliases for the user

Returns
Boolean — A boolean indicating whether the value was successfully set.

setCustomUserAttribute()method 
public function setCustomUserAttribute(key:String, value:String):Boolean

Sets a boolean custom user attribute. This can be any key/value pair and is used to collect extra information about the user. The key parameter is limited to 255 characters in length and can only contain alphanumeric characters and underscores. Keys which are too long will be truncated

Parameters

key:String — The identifier of the custom attribute
 
value:String — The value of the custom attribute

Returns
Boolean — A boolean indicating whether the value was successfully set.

public function setLanguage(language:String):Boolean

Sets the language of the user. Language Strings should be valid ISO 639-1 language codes. See https://www.loc.gov/standards/iso639-2/php/code_list.php. If not set here, user language will be inferred from the device locale.
