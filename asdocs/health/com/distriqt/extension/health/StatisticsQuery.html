withInterval()method 
public function withInterval(duration:int, timeUnit:int):StatisticsQuery

Set the duration and interval for each of the slices / buckets for the returned statistics. This specifies the time interval for each of the statistic results returned. So if you want daily totals you would use a duration of 1 and a unit of TimeUnit.DAYS
