Classpublic class Category
InheritanceCategory Object

A category represents a group of actions that can be applied to a notification by setting the category in the notification payload.

You should not construct this class directly but instead use the CategoryBuilder

A vector array of Action's to associate with this notification category. It should be no more than 3, and you should be aware that only the first 2 may be displayed in some situations.

public var identifier:String

A unique identifier for this category. This will be the value that you pass as part of your notification payload to indicate this category should be applied to the notification.
