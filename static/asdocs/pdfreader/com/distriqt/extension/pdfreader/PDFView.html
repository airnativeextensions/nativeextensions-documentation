dispose()method
public function dispose():void

Disposes the view, clearing any memory and objects used. Once called this view will be in an invalid state and will not respond to any further calls or events.

isSupported()method 
public function isSupported():Boolean

Whether displaying a view in your AIR application is supported on the current device. If this returns false but the main isSupported flag is true then calling show will launch an external viewer to show the dialog.

Returns
Boolean
load()method 
public function load():Boolean

Loads the pdf into the viewer.

If the file is remote (http etc) then this will start the download of the PDF and return true.

If the file is local or has already been downloaded then this will return false as the pdf is already loaded.

Returns
Boolean — true if the download was commenced and false if the file is already available
setPage()method 
public function setPage(page:int):Boolean

Change the currently displayed page to the specified page number.

The page number should be in the range 1 to totalPages

Parameters

page:int — The new page number to display

Returns
Boolean — true if the page change was performed successfully and false if the page number was invalid or if there was no document loaded in the view.

show()method 
public function show(resetState:Boolean = false):Boolean

Displays the PDF View to the user.

If the PDF is not yet loaded this will start the load and return false.

Parameters

resetState:Boolean (default = false) — If true the stored state of the PDF is cleared, i.e. current page, bookmarks etc.

Returns
Boolean — true if the display was successful
