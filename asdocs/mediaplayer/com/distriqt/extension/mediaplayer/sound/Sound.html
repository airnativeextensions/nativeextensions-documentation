Method Detail
loadFile()method
public function loadFile(file:File):Boolean

Load the specified file into the sound instance

Will dispatch Event.COMPLETE when loaded or IOErrorEvent.IO_ERROR if there was an error loading the file. It may also throw an Error if the file doesn't exist

Note: On Android if your file is located in the applicationStorage directory this will copy the file to a cached location for playback. AIR compresses media files in the APK and they need to be extracted before native players can access them.

Parameters

file:File — File reference to the sound file to load

Returns
Boolean — true if successful load was started
play()method 
public function play(loops:int = 0, sndTransform:SoundTransform = null):SoundChannel

Generates a new SoundChannel object to control the sound and initiates playback

Parameters

loops:int (default = 0) — Defines the number of times a sound loops back to the start before the sound channel stops playback. Set to -1 to loop continuously.
 
sndTransform:SoundTransform (default = null) — The initial SoundTransform object assigned to the sound channel

Returns
SoundChannel — A SoundChannel object, which you use to control the sound. This method returns null if there was an error initiating playback

Unloads the sound and releases any memory associated with this sound. After calling this the Sound instance will be invalid and will not playback sounds.
