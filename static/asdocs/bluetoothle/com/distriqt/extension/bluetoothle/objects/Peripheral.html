Packagecom.distriqt.extension.bluetoothle.objects
Interfacepublic interface Peripheral extends flash.events.IEventDispatcher

The Peripheral class represents a remote peripheral that was discovered by using the central manager scanning functionality.

You should not create an instance of this class directly but instead wait for discovered peripherals using the CentralManager.scanForPeripherals and listening for PeripheralEvent.DISCOVERED events.

subscribeToCharacteristic()method 
public function subscribeToCharacteristic(characteristic:Characteristic):Boolean

Subscribe to updates on this particular peripheral's characteristic. The Peripheral will be notified of your subscription and will send updates when it deems appropriate.

You should listen for the CharacteristicEvent.UPDATE event to receive updated characteristic values

The CharacteristicEvent.SUBSCRIBE and CharacteristicEvent.SUBSCRIBE_ERROR events indicate if the subscription to the characteristic was successful or not

Parameters

Boolean — true if the subscription attempt was started and false if the adapter is in an invalid state

writeValueForCharacteristic()method 
public function writeValueForCharacteristic(characteristic:Characteristic, value:ByteArray):Boolean

Attempt to write a value to the characteristic

The characteristic properties are checked to see if it contains the write or write without response properties and the appropriate write mode is selected. If it contains both then the write with response type is used. If neither is available this function will return false.

Parameters

characteristic:Characteristic — The peripheral characteristic to write to
 
value:ByteArray — The ByteArray data value to write to the peripheral's characteristic

Returns
Boolean — true if the write attempt was started and false if the adapter is in an invalid state or if the characteristic does not support writing
