### iOS

#### Dynamic Frameworks

This extension uses dynamic frameworks so you must include the frameworks (and any dependent swift libs) in your application for signing by AIR.

To do this create a `Frameworks` directory at the top / root level of your application and ensure it is packaged with your AIR application.

It should contain any of the files in the supplied `assets/ios/Frameworks` directory (including `dylib` and `framework` directories).
