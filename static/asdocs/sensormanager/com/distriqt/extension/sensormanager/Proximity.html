Whether proximity sensing is supported on the current device. This will return true if it is possible to start monitoring for proximity events.

proximityStateproperty 
proximityState:String  [read-only]

Retrieves the current proximity state. This will be one of the values defined in the ProximityState class.

For the proximity state to be valid you must currently be monitoring for proximity changes by having called startMonitoring
