STYLE_ANNULARConstant 
public static const STYLE_ANNULAR:String = annular

A circular style progress dialog with a progress bar that can be updated to show a processes progress.

To be used with the ProgressDialogBuilder.

Supported on iOS only. On Android this will be the equivalent of STYLE_DETERMINATE

See also

com.distriqt.extension.dialog.builders.ProgressDialogBuilder.setStyle()
STYLE_DETERMINATEConstant 
public static const STYLE_DETERMINATE:String = determinate

A determinate style progress dialog with a progress indicator that can be updated to show the progress of a process.

To be used with the ProgressDialogBuilder.

Under Android this will display a horizontal progress bar. Under iOS this will display a circular progress bar.

See also

com.distriqt.extension.dialog.builders.ProgressDialogBuilder.setStyle()
