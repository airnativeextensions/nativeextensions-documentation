BEFORE_DISPLAYEDConstant
public static const BEFORE_DISPLAYED:String = braze_iam_before_displayed

Dispatched before an in-app message is displayed You can modify the message during this event. You must call displayInAppMessage() to present the message. If you don't add a listener for this event then the message will be automatically shown.
