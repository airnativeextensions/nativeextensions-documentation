public function enableMessaging(enabled:Boolean):Boolean

If you have disabled messaging via a configuration setting (eg in the manifest additions) calling this will enable messaging on the device. You may wish to do this if you'd like to give users more control over the data they send.

messageDisplaySuppressed()method 
public function messageDisplaySuppressed(suppressed:Boolean):Boolean

Temporarily disable messages. You should suppress messages whenever you do not wish your user to be interrupted, for example to avoid interrupting a sequence of payment processing screens. You should disable this after the process has completed.

Parameters

suppressed:Boolean — true to temporarily suppress messages, and false to re-enable messages.

Returns
Boolean — true if the call was successful and false if the current service doesn't support this functionality.
