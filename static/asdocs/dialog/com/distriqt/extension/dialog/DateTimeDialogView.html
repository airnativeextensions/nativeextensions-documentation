Packagecom.distriqt.extension.dialog
Interfacepublic interface DateTimeDialogView extends DialogView, flash.events.IEventDispatcher

Represents a Date / Time Dialog view presented with the Dialog.service.create functionality.

These dialogs are presented modally with your application being in the background. You specify the options of the dialog by creating using the DateTimeDialogBuilder and passing it to the Dialog.service.create() function.

Events are dispatched from the dialog, including change, selection and cancel events. The normal DialogEvent.DIALOG_CANCELLED event is fired when the dialog is cancelled by the user. The changed and selection events are in the DialogDateTimeEvent class.

  • MODE_TIME: Displays only the time selection in the picker dialog (hour, minute)
  • MODE_DATE: Displays only the date selection in the picker dialog (year, month, day)
  • MODE_DATE_AND_TIME: Displays both the date and time selection in the picker dialog simultaneously. iOS only
Android

Please note that on Android you can only display either the date or the time, so you'll have to accommodate this in your code. A simple option is to create both a date and time picker and the user will have to interact with both before returning to your application.
