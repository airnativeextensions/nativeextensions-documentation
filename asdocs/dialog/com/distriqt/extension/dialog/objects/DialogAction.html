STYLE_CANCELConstant
public static const STYLE_CANCEL:int = 1

The style is used for the cancel actions. Generally only one is allowed and it is normally uses bold text or locating the button in a slightly different location.

STYLE_DESTRUCTIVEConstant 
public static const STYLE_DESTRUCTIVE:int = 2

This style indicates a destructive action, will make the button stand out either being coloured red or in a slightly different location, depending on the operating system and version.

Not supported in iOS < 8. If not supported the button will be treated as default.

STYLE_NEUTRALConstant 
public static const STYLE_NEUTRAL:int = 4

This style indicates a neutral action, will make the action located near the positive and cancel actions potentially separate from the other actions.

Not supported in iOS. If not supported the button will be treated as default.

STYLE_POSITIVEConstant 
public static const STYLE_POSITIVE:int = 3

Not supported in iOS. If not supported the button will be treated as default.
