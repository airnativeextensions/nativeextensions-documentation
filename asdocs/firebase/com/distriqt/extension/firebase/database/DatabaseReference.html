Packagecom.distriqt.extension.firebase.database
Interfacepublic interface DatabaseReference extends Query, flash.events.IEventDispatcher

A Firebase reference represents a particular location in your Database and can be used for reading or writing data to that Database location.

This class is the starting point for all Database operations. After you've initialized it with a URL, you can use it to read data, write data, and to create new DatabaseReferences.

public function push():DatabaseReference

Create a reference to an auto-generated child location.

The child key is generated client-side and incorporates an estimate of the server's time for sorting purposes. Locations generated on a single client will be sorted in the order that they are created, and will be sorted approximately in order across all clients.

public function setValue(value:*, priority:String = null):Boolean

Set the data and priority to the given values.

Priorities are an optional (alphanumeric) value of each node, which is used to sort the children under a specific parent or in a query if no other sort condition is specified. The priority of a node is hidden from most views of the data.

updateChildren()method 
public function updateChildren(update:Object):Boolean

Update the specific child keys to the specified values. Passing null in as a value to updateChildren() will remove the value at the specified location.

To simplify the creation of the update parameter we suggest using the UpdateChildrenBuilder

Simultaneous updates made this way are atomic: either all updates succeed or all updates fail.

Parameters

update:Object — The update(s) to apply to the children of this database reference

Returns
Boolean — true if successfully started and false if the extension is in an invalid state or if the value provided is an unsupported type

See also

com.distriqt.extension.firebase.database.builders.UpdateChildrenBuilder
com.distriqt.extension.firebase.database.events.DatabaseReferenceEvent.UPDATE_CHILDREN_COMPLETE
com.distriqt.extension.firebase.database.events.DatabaseReferenceEvent.UPDATE_CHILDREN_ERROR

Example 
var ref:DatabaseReference = FirebaseDatabase.service.getReference( "test" );
                
ref.updateChildren( 
    new UpdateChildrenBuilder()
        .update( "/children_test/numericValue", 100 )
        .update( "/children_test/stringValue", "some value" )
        .build()
);
