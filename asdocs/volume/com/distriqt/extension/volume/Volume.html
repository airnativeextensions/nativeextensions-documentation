Classpublic final class Volume
InheritanceVolume flash.events.EventDispatcher

This class represents the Volume extension.

Stream

Streams represent the different types of sound that can be generated from a device, inlcuding ring volumes, notifications, alarms and music. Most of the time you'll only be interested in the default music stream.

On iOS there is only the music stream available to developers.

Android gives access to all the streams in the VolumeStream class.

Volume Level

You can retrieve and set the volume level using the getVolume and setVolume functions. 

     var volume:Number = Volume.service.getVolume();
     
     Volume.service.setVolume( 0.75 );
Change events

Listening to volume change events is achieved by calling register specifying the stream of interest. Volume events are dispatched when either the user changes the volume using the native controls, or by pressing the volume buttons, or when you change the volume using the setVolume function. 

     Volume.service.addEventListener( VolumeEvent.CHANGED, volumeChangedHandler );
     Volume.service.register();
          ...
          private function volumeChangedHandler( event:VolumeEvent ):void
     {
         trace( event.type + "::" +event.volume + " [" + event.stream + "]" );
     }

You should make sure you call unregister when you are done listening to cleanup and remove any volume listening functions.
