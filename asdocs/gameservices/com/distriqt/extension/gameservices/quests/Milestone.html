Property Detail
currentProgressproperty
public var currentProgress:Number

The current progress of the targetProgress required to complete the milstone.

When the milestone state is STATE_CLAIMED or STATE_COMPLETED_NOT_CLAIMED the value of targetProgress is returned.

When the milestone state is STATE_NOT_COMPLETED the return value is the number of increments to the event associated with the milestone that have occurred since the quest was accepted.

When the milestone state is STATE_NOT_STARTED the return value is 0.

eventIdproperty 
public var eventId:String

The ID of the associated event.

Increments to this event will increase the user's progress toward this milestone when the milestone is in the STATE_NOT_COMPLETED state.

rewardDataproperty 
public var rewardData:ByteArray

The completion reward data for this milestone. Generally you should use this data to reward the player when they 'claim' the quest, i.e. on a QuestEvent.QUEST_CLAIM_SUCCESS event.
