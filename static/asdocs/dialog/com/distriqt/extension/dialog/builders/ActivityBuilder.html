setCancelable()method 
public function setCancelable(cancelable:Boolean):ActivityBuilder

If true then the dialog can be cancelled by the user clicking the background or outside the dialog. This allows a user to cancel an operation.

Alternatively if this is false then user interaction can not stop the indictor and you should use dismiss when you need to remove the dialog

Parameters

cancelable:Boolean

setIdentifier()method 
public function setIdentifier(id:int):ActivityBuilder

Sets a requested value for the identifier (id) of the created dialog. Note that if a dialog of the same id already exists and hasn't been disposed then this value will be ignored and a valid value assigned to the view on creation.

You can check the id through DialogView.id

Parameters

id:int — The requested id to assign to the view

Returns

setStyle()method 
public function setStyle(style:String):ActivityBuilder

This sets the style of the activity indicator animation, the DialogType.STYLE_SPINNER uses a classic spinner style animation with marks around a circle, whereas DialogType.STYLE_ANNULAR uses a circle line animation.

Ignored on Android: Android always presents the native indicator which is an annular style activity indicator on modern versions of Android.

Parameters

style:String — The style of the activity indicator, defaults to DialogType.STYLE_ANNULAR

Returns
