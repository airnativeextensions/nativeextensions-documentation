storeDeterminedEligibleproperty 
public var storeDeterminedEligible:Boolean = false

Indicates whether the current user is determined to be eligible for the offer by the source store. If this value is true then you can assume the user is eligible for the offer. If it is false then you will need to use the process defined by the current store to determine user eligibility. For example, Google Play Billing only provides details on these offers to users who are eligible for the offer and so this value will be true. As opposed to Apple In-App Purchases where you must determine eligibility by determining if the user has previously purchased a subscription.
