Property Detail
canWriteproperty
canWrite:Boolean  [read-only]

Checks if the specified app can modify system settings. As of API level 23, an app cannot modify system settings unless it declares the Manifest.permission.WRITE_SETTINGS permission in its manifest, and the user specifically grants the app this capability. To prompt the user to grant this approval call showManageSettingsScreen() Reference
