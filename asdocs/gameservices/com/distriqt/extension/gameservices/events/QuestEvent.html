QUEST_ACCEPT_SUCCESSConstant 
public static const QUEST_ACCEPT_SUCCESS:String = quests:quest:accept:success

Dispatched when an accept call completes successfully. The player will now have an active quest, which will be the only Quest object in the quests array.

QUEST_CLAIM_ERRORConstant 
public static const QUEST_CLAIM_ERROR:String = quests:quest:claim:error

Dispatched when the quests claim operation fails.

errorCode and message will contain details on the error that occurred.

QUEST_CLAIM_SUCCESSConstant 
public static const QUEST_CLAIM_SUCCESS:String = quests:quest:claim:success

Dispatched when a claim call completes successfully. The player has now claimed the reward for quest, which will be the only Quest object in the quests array.

QUEST_COMPLETEDConstant 
public static const QUEST_COMPLETED:String = quests:quest:completed

Dispatched when the current player completes a quest. You must have called listenForCompletion in order for this event to be dispatched.

See also

com.distriqt.extension.gameservices.quests.Quests.listenForCompletion()
QUESTS_LOAD_ERRORConstant 
public static const QUESTS_LOAD_ERROR:String = quests:load:error

Dispatched when the quests load operation fails.

errorCode and message will contain details on the error that occurred.

QUESTS_LOAD_SUCCESSConstant 
public static const QUESTS_LOAD_SUCCESS:String = quests:load:success

Dispatched when the quests load operation is completed successfully. quests will contain an array of Quest objects representing the available quests for the current player.
