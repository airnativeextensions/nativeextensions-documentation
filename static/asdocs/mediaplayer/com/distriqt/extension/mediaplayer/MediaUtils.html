generateThumbnail()method
public function generateThumbnail(path:String, width:int = 425, height:int = 355, time:Number = 0):void

Generates a thumbnail from the content of the video.

This will only work on local video files.

Parameters

path:String — The path to the file to play (this should be the native path to a file)
 
width:int (default = 425) — The maximum width of the generated thumbnail in pixels
 
height:int (default = 355) — The maximum height of the generated thumbnail in pixels
 
time:Number (default = 0) — The time in seconds to use to generate the thumbnail
