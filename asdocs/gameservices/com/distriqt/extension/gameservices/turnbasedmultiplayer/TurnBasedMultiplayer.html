displayCreateMatchUI()method 
public function displayCreateMatchUI(options:PlayerSelectionOptions):void

This is the start of the UI to create a game. The process works slightly differently on the platforms however this is always the starting point and will end in a create game event.

Google Play

Displays a user interface that allows the player to select from other players and invite them into a match. You can control the maximum and minumum number of players to select and other options by passing in the PlayerSelectionOptions instance.

This will fire a TurnBasedMultiplayerEvent event with the details of the player selection. You should then call createMatch with the details selected by the user or modify if your game requires it.

Game Center

In Game Center this displays a create match user interface. This allows the user to select players and create a match.

This will fire a TurnBasedMatchEvent with either create success or fail information.

Parameters

options:PlayerSelectionOptions — The options for the selection UI
