Packagecom.distriqt.extension.location
Classpublic final class Location
InheritanceLocation flash.events.EventDispatcher

This class represents the Location extension. You use this extension to monitor for entry and exit of geographic regions and to track the user location. It uses all the available device location methods, including GPS, and network location.

Geofences

A geofence is nothing more than an virtual boundary that corresponds to an actual location in space. It is defined by a geographic position and a distance or radius.

checkLocationSettings()method 
public function checkLocationSettings(request:LocationRequest = null):Boolean

On Android you have the ability to ask the user to change the location services directly from your application.

Calling this function with your LocationRequest will check if the device currently has the correct services enabled to support the location request. The major factor here is the priority of the request which specifies which services will need to be enabled.

During this process there may be a dialog presented to the user to ask to enable the required services, if they currently are disabled.

Parameters

request:LocationRequest (default = null) — The location request to check settings for

Returns
Boolean — true if the current device platform and version supports the settings check and false if not.

hasAuthorisation()method 
public function hasAuthorisation():Boolean

Checks the authorisation status of the application, to see if the user has previously denied access to location services.

This will return true, if the user has allowed access, either only while in use or always. If the user has not yet been asked to grant permission, or if the user has previously denied access it will return false.

Returns
Boolean — true if the user has granted access and false otherwise

isAvailable()method 
public function isAvailable():Boolean

Checks to see if the location services (GPS etc) are available on the device. The user may have disable location services and you should respect this setting by not using location services in your application, or by presenting information about why they should enable location services.

Returns
Boolean — true if the location providers are available, i.e. either the network or GPS location provider is available to be used. false if neither is not available.
requestAuthorisation()method 
public function requestAuthorisation(status:String):Boolean

Request access to location services. Performs an operation to force the display of the permission dialog.

Note on iOS this will only be shown if the authorisation status is currently NOT_DETERMINED

This allows you to control when the permission dialog is presented.
