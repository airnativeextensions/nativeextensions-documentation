public function setIdTokenWithRawNonce(idToken:String, rawNonce:String):OAuthCredential

Adds an ID token and raw nonce to the credential being built. The raw nonce is required when an OIDC ID token with a nonce field is provided. The SHA-256 hash of the raw nonce must match the nonce field in the OIDC ID token.
