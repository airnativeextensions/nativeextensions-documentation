currencyCodeproperty
public var currencyCode:String

A currency code is a three-letter code that is, in most cases, composed of a country’s two-character Internet country code plus an extra character to denote the currency unit. For example, the currency code for the Australian dollar is “AUD”. Currency codes are based on the ISO 4217 standard.

paymentModeproperty 
public var paymentMode:String

The payment mode for this product discount. The payment mode indicates how the product discount price is charged:

  • One or more times, for PAYMENTMODE_PAY_AS_YOU_GO mode
  • Once in advance, for PAYMENTMODE_PAY_UP_FRONT mode
  • No initial charge, for PAYMENTMODE_FREE_TRIAL mode
Use the payment mode to display an accurate description of the product discount in your UI.

storeDeterminedEligibleproperty 
public var storeDeterminedEligible:Boolean = false

Indicates whether the current user is determined to be eligible for the discount by the source store. If this value is true then you can assume the user is eligible for the discount. If it is false then you will need to use the process defined by the current store to determine user eligibility. For example, Google Play Billing only provides details on these offers to users who are eligible for the discount and so this value will be true. As opposed to Apple In-App Purchases where you must determine eligibility by determining if the user has previously purchased a subscription.
