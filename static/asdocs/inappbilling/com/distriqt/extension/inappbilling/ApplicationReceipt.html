public function refresh(options:ApplicationReceiptProperties = null, onComplete:Function = null):Boolean

This will refresh the application receipt value, populating it with the most recent values. Events Calling this function may potentially dispatch any of the following:

  • ApplicationReceiptEvent.REFRESH_SUCCESS
  • ApplicationReceiptEvent.REFRESH_FAILED

Parameters

options:ApplicationReceiptProperties (default = null) — Debugging options
 
onComplete:Function (default = null) — Function called when refresh operation is complete, function c(success:Boolean, error:Error ):void

Returns
Boolean — true if the refresh was successfully started

function refreshSuccessHandler( event:ApplicationReceiptEvent ):void
{
    var appReceipt:String = InAppBilling.service.applicationReceipt.getAppReceipt();
    trace( "getAppReceipt() = " + appReceipt );
}
function refreshFailedHandler( event:ApplicationReceiptEvent ):void
{
    trace( "application receipt refresh failed" + event.error );
}
