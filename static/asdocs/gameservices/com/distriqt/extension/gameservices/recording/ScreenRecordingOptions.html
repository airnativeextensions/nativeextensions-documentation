Property Detail
filepathproperty
public var filepath:String

If the screen recorder supports recording to an application accessible file then you can specify the path of the file here. If empty the extension will create a default filename in the documents directory.

microphoneEnabledproperty 
public var microphoneEnabled:Boolean = false

Unsupported

qualityproperty 
public var quality:String = high_quality

Set the quality of the recording on supported services. If supported this will set the quality of the recorded video. Should be one of the following values:

  • ScreenRecordingOptions.LOW
  • ScreenRecordingOptions.MEDIUM
  • ScreenRecordingOptions.HIGH

uiOverlayEnabledproperty 
public var uiOverlayEnabled:Boolean = true

uiOverlayStopButtonproperty 
public var uiOverlayStopButton:BitmapData = null

If the UI Overlay is in use, this bitmap data asset will be used as the stop button. If null then the default asset will be used
