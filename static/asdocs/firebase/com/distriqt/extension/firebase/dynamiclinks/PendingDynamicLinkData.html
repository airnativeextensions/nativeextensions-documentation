minimumAppVersionproperty 
public var minimumAppVersion:String

The minimum application version that supports the Dynamic Link. This is retrieved from the imv= parameter of the Dynamic Link URL. Note: This is not the minimum iOS or Android system version, but the minimum app version. If app version of the opening app is less than the value of this property, then app expected to open the store to allow user to download most recent version. App can notify or ask user before opening AppStore/PlayStore.
