sharePost()method 
public function sharePost(post:SocialPost, network:String):Boolean

This is similar to the normal share action except it uses the built in social network services, rather than the generic share activity.

This is mainly for iOS using the Social framework introduced on iOS 6 and the SLComposeViewController

Parameters

post:SocialPost — The SocialPost instance to post to the specified network
 
network:String — The network to post to

Returns
Boolean — true if successful and false otherwise

Example
A simple share to Facebook if available 
         if (Share.service.social.isNetworkSupported( SocialNetwork.FACEBOOK ))
         {
             Share.service.social.sharePost( 
                 
                 new SocialPostBuilder()
                     .setInitialText( "Awesome native extensions" )
                     .addURL( "http://airnativeextensions.com" )
                     .build(),
                     
                 SocialNetwork.FACEBOOK 
             );
         }
