canAuthenticateWithBiometryType()method 
public function canAuthenticateWithBiometryType(biometryType:int):Boolean

Whether the current device supports authentication using fingerprints

Parameters

biometryType:int (default = NaN) — The biometry type to check for. Set to ANY to check for any available authentication type.

Returns
Boolean — true if the (or any) biometric authentication method is available and false if not, i.e. there are no fingerprints stored or there is no fingerprint reader on the device or there is no facial recognition method.
