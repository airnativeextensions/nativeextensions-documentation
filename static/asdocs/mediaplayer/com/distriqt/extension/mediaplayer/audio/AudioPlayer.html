Packagecom.distriqt.extension.mediaplayer.audio
Interfacepublic interface AudioPlayer extends flash.events.IEventDispatcher

Represents an audio player interface with no application UI.

You can use this to control playback of an audio file in your application using the native player, and enabling advanced features such as background audio playback.

destroy()method
public function destroy():void

Destroys the player, clearing any caches and removing the player from memory.

Any calls to the player after calling destroy will have undefined results.

load()method 
public function load(source:String):Boolean

Once a player has been created you can change the loaded media by using this call, passing in the path to the new file.

Note: this function may return true even if the file couldn't be loaded, especially for remote files. The return value of true only indicates that the source from which the audio player will load has been changed.

Parameters

source:String — The source of the audio to play (this can be the native path to a file or a url to a remote file)

Returns
Boolean — true if successfully changed and false if the player wasn't yet created.
loadFile()method 
public function loadFile(file:File):Boolean

Once a player has been created you can change the loaded media by using this call, passing in the new file.

Note: On Android if your file is located in the applicationStorage directory this will copy the file to a cached location for playback. AIR compresses media files in the APK and they need to be extracted before native players can access them.

Parameters

file:File — The source File of the audio to play

Returns
Boolean — true if successfully changed and false if the player wasn't yet created.

seek()method 
public function seek(time:Number):Boolean

Seeks to the specified time in the media. The time must be less than the total duration of the media, otherwise nothing is done.

Parameters

time:Number — The time to seek to (seconds)

Returns
Boolean — true if the seek was started successfully, and false if there was an error
setPlaybackSpeed()method 
public function setPlaybackSpeed(rate:Number):Boolean

Control the playback speed of the audio.

You must have set the playbackSpeedEnabled option when creating this player in order for this to work correctly.

Parameters

rate:Number — The playback rate, 0.5 for half speed to 2.0 for double speed

Returns
Boolean — true if the playback speed change was successful, and false if there was an error or if playback speed is not enabled for this player

See also

com.distriqt.extension.mediaplayer.audio.AudioPlayerOptions.playbackSpeedEnabled
setVolume()method 
public function setVolume(volume:Number):Boolean

The playback volume for the audio player, ranging from 0.0 through 1.0 on a linear scale.

A value of 0.0 indicates silence; a value of 1.0 (the default) indicates full volume for the audio player instance.

Parameters

volume:Number

Returns
Boolean — true if the volume change was successful, and false if there was an error
