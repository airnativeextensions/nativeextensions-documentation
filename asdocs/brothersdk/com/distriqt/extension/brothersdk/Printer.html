setAddress()method 
public function setAddress(address:String):Printer

The address is used differently depending on the channel:

  • Channel.WIFI: It is the ip address of the printer
  • Channel.BLUETOOTH: It is the macAddress of the printer
  • Channel.BLUETOOTH_LOW_ENERGY: It is the local name of the printer

public function setChannel(channel:String):Printer

Sets the channel to use to connect to the printer, one of the values in the Channel class. This controls whether to use WiFi, Bluetooth etc to connect to the printer.
