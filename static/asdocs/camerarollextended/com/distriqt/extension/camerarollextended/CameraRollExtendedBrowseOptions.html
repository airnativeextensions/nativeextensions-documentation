useNativePickerproperty 
useNativePicker:Boolean

Android Only

If true the user will be presented with the native picker dialog. This will allow the user to select an application to pick images. There is no maximum count currently available on this, and in fact some older versions of Android and some applications will only allow one image to be selected.

The advantage of the native picker on Android though is that the user will be able to select content from any of the application they have that provide content, eg. Google Photos, Drive, or simply the Gallery

If this is false, a custom picker dialog will be presented. This picker will appear inside your application and allow multiple selection and control of the maximum number of images.
