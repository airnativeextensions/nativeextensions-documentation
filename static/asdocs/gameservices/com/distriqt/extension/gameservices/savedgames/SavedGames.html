Interfacepublic interface SavedGames extends flash.events.IEventDispatcher

Saved Games

This interface defines the saved games functionality for the Game Services native extension.

All the functionality here can be accessed through the main GameServices class as below: 

     GameServices.service.savedGames.[FUNCTION]
Android

Additional setup is required to make saved games work. You'll need to make sure you've enabled "Saved Games" in the Developer console for your game services application, and also add the Drive API to the linked API console project.

displaySavedGamesUI()method 
public function displaySavedGamesUI(title:String, allowAdd:Boolean = false, allowDelete:Boolean = false, maxSavedGames:int = -1):Boolean

Displays a native UI to the user allowing the user to select a save game.

Parameters

title:String — The title to display in the UI
 
allowAdd:Boolean (default = false) — Whether or not to display a create new saved game option in the UI
 
allowDelete:Boolean (default = false) — Whether or not to provide a delete option for each saved game in the UI
 
maxSavedGames:int (default = -1) — The maximum number of snapshots to display in the UI (Set to -1 to have no limit)

Returns
Boolean — true if the UI was displayed or false if there is no UI available
loadSavedGames()method 
public function loadSavedGames(forceReload:Boolean = false):void

Loads the players available saved games and returns a GameServicesSavedGameEvent.SAVEDGAMES_LOADED event with an array of SavedGame objects.

Parameters

forceReload:Boolean (default = false) — If true this will contact the server for updated information otherwise the locally synced cache is used.


Example

The following starts the load of available player saved games. 

         GameServices.service.savedGames.addEventListener( GameServicesSavedGameEvent.SAVEDGAMES_LOADED,     savedGamesLoadedHandler );
         GameServices.service.savedGames.addEventListener( GameServicesSavedGameEvent.SAVEDGAMES_LOAD_ERROR, savedGamesLoadErrorHandler );
                  if (GameServices.service.isSignedIn())
         {
             GameServices.service.savedGames.loadSavedGames();
         }
