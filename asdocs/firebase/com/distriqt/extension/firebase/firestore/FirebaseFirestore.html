disableNetwork()method 
public function disableNetwork():void

Disables network access for this instance.

While the network is disabled, any snapshot listeners or get() calls will return results from cache, and any write operations will be queued until network usage is re-enabled via a call to enableNetwork().

dispose()method 
public function dispose():void

Disposes the extension and releases any allocated resources. Once this function has been called, a call to init is neccesary again before any of the extensions functionality will work.

disposeUnusedReferences()method 
public function disposeUnusedReferences():void

During the course of using the Firebase Firestore extension you will be creating and destroying references and collections that may not get immediately disposed.

If you need to manage memory and remove some items you can call this function to dispose any references and queries that have been created and do not have active event listeners.

You should make sure that any references you are holding onto either have an event listener or are released after calling this function.
