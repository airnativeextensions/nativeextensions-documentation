Property Detail
currencyCodeproperty
public var currencyCode:String

A currency code is a three-letter code that is, in most cases, composed of a country’s two-character Internet country code plus an extra character to denote the currency unit. For example, the currency code for the Australian dollar is “AUD”. Currency codes are based on the ISO 4217 standard.

currencySymbolproperty 
public var currencySymbol:String

The string used by the receiver as a local currency symbol. Due to some inconsistencies in the billing platforms this value may not be consistent across different localities. It is suggested you use the priceString wherever possible

subscriptionPeriodproperty

If available this will contain information on the period of the subscription product. It will be null for non-subscription products and for services that do not report this information about subscription products.
