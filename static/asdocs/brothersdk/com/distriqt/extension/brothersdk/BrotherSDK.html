startSearchBluetoothPrinter()method 
public function startSearchBluetoothPrinter(callback:Function = null):void

Starts a search for paired bluetooth printer devices. On completion the BrotherSDKSearchEvent.RESULT event will be dispatched and the optional callback function will be called. The event and the callback contain a list of the discovered printers as a parameter: printers:Vector.<Printer>

Parameters

callback:Function (default = null) — Optional callback function with the signature: function( printers:Vector.<Printer> ):void
