Property Detail
isSupportedproperty
isSupported:Boolean  [read-only]

Returns true if the settings functionality is supported on the current device. This will return false on platforms that don't support settings or that currently have not been implemented.

Additionally on Android it will return false if you haven't added the correct manifest additions and custom resources to your application.
