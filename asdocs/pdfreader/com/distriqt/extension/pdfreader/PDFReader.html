This class represents the PDF Reader extension.

The PDF Reader allows you to display an interactive PDF view to your user.

The extension provides functionality to display a smooth and responsive PDF view in your application complete with:

  • iBooks-like document navigation
  • Device rotation and all orientations
  • Encrypted (password protected) PDFs
  • PDF links (URI and go to page)
  • PDFs with rotated pages

This implementation has been tested with large PDF files.

public function createView(options:PDFViewOptions):PDFView

Create an instance of a PDFView, to display a PDF in a view.

Parameters

options:PDFViewOptions — The options for the created view. We recommend using the PDFViewBuilder to generate this object.

Returns
PDFView — An instance of a PDFView or null if not supported.

Example
Create a PDFView and show it to the user 
         var path:String = File.applicationDirectory.resolvePath( "document.pdf" ).nativePath;
         var view:PDFView = PDFReader.service.createView( 
                             new PDFViewBuilder()
                                 .setPath( path )
                                 .build()
                             );
         view.show();

Sets which of the controls will be visible when a PDF is displayed to the user. These are the controls along the top bar in the viewer.

If you do not call this function then all the controls are shown.

Parameters

Examples

This simple example shows the basic functionalty of how to display a PDF in the reader. The PDF file is a file named "Reader.pdf" and packaged in the application directory with the application. 

     //
     // Initialise the extension
     PDFReader.init( DEV_KEY );
     if (PDFReader.isSupported)
     {
         //
         // Show a PDF
         var path:String = File.applicationDirectory.resolvePath( "document.pdf" ).nativePath;
         PDFReader.service.createView( 
                         new PDFViewBuilder()
                             .setPath( path )
                             .build()
                         ).show();
     }
