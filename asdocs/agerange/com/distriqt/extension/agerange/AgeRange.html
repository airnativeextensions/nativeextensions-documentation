isEligibleForAgeRange()method 
public function isEligibleForAgeRange(successCallback:Function, errorCallback:Function = null):void

Checks if the currently signed in user is eligible for age range data. Currently only supported on iOS 26.2+. Unsupported platforms will return a "Not supported" error with code "-1".

Parameters

successCallback:Function — A callback function which receives a Boolean indicating if the user is eligible. Should have the signature: function( eligible:Boolean ):void
 
errorCallback:Function (default = null) — A callback function which is called if there was an error performing the eligibility check. Should have the signature: function( error:Error ):void

requestAgeRange()method 
public function requestAgeRange(options:AgeRangeRequest = null, successCallback:Function = null, errorCallback:Function = null):void

Performs an age signals check with Google Play for the currently signed in user.

Parameters

options:AgeRangeRequest (default = null) — An AgeRangeRequest object containing the options for the age signals check.
 
successCallback:Function (default = null) — A callback function which receives the result of the age signals check as an AgeSignalsResult object. Should have the signature: function function( result:AgeSignalsResult ):void
 
errorCallback:Function (default = null) — A callback function which is called if there was an error performing the age signals check. Should have the signature: function function( error:Error ):void
