PURCHASE_SUCCESSConstant

Deprecated: Please Use PurchaseEvent.PURCHASES_UPDATED

A purchase was successful. When you receive this event you should deliver the product to the application.

You will receive this as the result of a restore, and after a purchase was made, for each individual successful purchase.

The #data contains an array with one element, being the details of the purchase.

PURCHASES_UPDATEDConstant 
public static const PURCHASES_UPDATED:String = purchases:updated

Dispatched whenever a purchase state has changed. This can be when the purchases queue is updated, when a purchase transaction is complete or when a user cancels a purchase.

You should use this event to iterate over the Purchase objects contained in the data array using the Purchase.transactionState to process the purchase state. 

         private function purchasesUpdatedHandler( event:PurchaseEvent ):void
         {
          for each (var purchase:Purchase in event.data)
          {
              switch (purchase.transactionState)
              {
                  case case Purchase.STATE_FAILED:
                      break;
                           case Purchase.STATE_PURCHASED:
                      break;
                           // ETC...
              }
          }
         }
