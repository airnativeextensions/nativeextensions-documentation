saveFilesInCacheproperty 
public var saveFilesInCache:Boolean = true

By default the extension will save the captured images to the application cache directory. This means that the files may be removed by the OS at certain points.

If you set this to false the files will be saved into the applications documents directory, making the files permanent. You can delete them manually (using File) if you require.

saveToCameraRollproperty 
public var saveToCameraRoll:Boolean = true

If true then the media is saved to the default location for images on the device and if necessary added to the user's device media.

If false then the media will be saved to a private application location and not added to the user's device media.
