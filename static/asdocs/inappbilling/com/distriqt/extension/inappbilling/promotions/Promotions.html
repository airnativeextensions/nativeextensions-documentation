showCodeRedemptionDialog()method 
public function showCodeRedemptionDialog():Boolean

Shows the system dialog to enter a promotion code for a user to redeem a code for a subscription or product. If the code is valid, a purchase will be added to the queue for you to process. You should make sure you have added a listener for the `PurchaseEvent.PURCHASES_UPDATED` before calling this.

Returns
Boolean — true if the dialog was shown and false if the dialog is not supported.
