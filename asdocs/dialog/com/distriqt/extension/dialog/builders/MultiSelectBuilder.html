setCancelable()method 
public function setCancelable(cancelable:Boolean = true):MultiSelectBuilder

If true then the dialog can be cancelled by the user clicking the background or outside the dialog. This allows a user to cancel an operation.

Alternatively if this is false then user interaction cannot cancel the dialog and they must select one of the available options.

Parameters

cancelable:Boolean (default = true) — true if the dialog is cancelable or false if the dialog requires interaction

Returns

setIdentifier()method 
public function setIdentifier(id:int):MultiSelectBuilder

Sets a requested value for the identifier (id) of the created dialog. Note that if a dialog of the same id already exists and hasn't been disposed then this value will be ignored and a valid value assigned to the view on creation.

You can check the id through DialogView.id

Parameters
