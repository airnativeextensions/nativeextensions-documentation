Interfacepublic interface Tracker extends flash.events.IEventDispatcher

Composes and sends hits to Google Analytics

A Tracker must be initialized with a tracking id of an app profile before you can send any hits. Use GoogleAnalytics.service.getTracker( trackingId ) to get an instance of this object.

setCampaignParamsOnNextHit()method 
public function setCampaignParamsOnNextHit(url:String):void

Includes the campaign parameters contained in the URI referrer in the next hit. If there is no referrer, or the referrer does not contain campaign parameters, this method does not add anything to the next hit.

Use this method to track in-app events driven by advertising with deep linking campaigns.

Valid campaign parameters are:

  • utm_id
  • utm_campaign
  • utm_content
  • utm_medium
  • utm_source
  • utm_term
  • dclid
  • gclid

Parameters

url:String


Example 
         tracker.setCampaignParamsOnNextHit( "http://my.site.com/index.html?referrer=utm_source%3Dsource%26utm_campaign%3Dwow" );
