MATCH_REMOVEDConstant 
public static const MATCH_REMOVED:String = turnbasedmultiplayer:match:removed

Dispatched when a match has been removed from the local device. For example, this might occur if the player leaves the match on another device.

data will be null and message will contain the matchId of the match that was removed from the device.

MATCH_UPDATEDConstant 
public static const MATCH_UPDATED:String = turnbasedmultiplayer:match:updated

Dispatched when a new update to a match arrives.

data will contain the updated match information.

When this event occurs it is important that you assess the updated match status and update your game accordingly.
