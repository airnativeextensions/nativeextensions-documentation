This class represents the ExpansionFiles native extension.

Firstly you'll need to acquire your license key, a Base64-encoded RSA public key for your application. You'll need to pass this to the setup function.

  • Setting Up for Licensing: http://developer.android.com/google/play/licensing/setting-up.html

To use and attach expansion files to an APK, you must at least have published an APK to the alpha test channel:

  • Use and attach APK expansion files: https://support.google.com/googleplay/android-developer/answer/2481797?hl=en

setup()method 
public function setup(key:String, salt:Vector.<int>):void

This function must be called to setup the licensing and salt values for your application.

The key value is a string that is the Base64-encoded RSA public key for your publisher account, available from the profile page on the Developer Console (see Setting Up for Licensing).

The salt value is an array of random int values that the licensing Policy uses to create an Obfuscator. The salt ensures that your obfuscated SharedPreferences file in which your licensing data is saved will be unique and non-discoverable.

Parameters

key:String — Base64 encoded RSA public key for your licensing publisher account
 
salt:Vector.<int> — An array of integers to use as the obfuscation salt value
