Property Detail
acceptedTimestampproperty
public var acceptedTimestamp:Number = -1

The timestamp (in milliseconds since epoch) at which this quest was accepted by the player. If the quest has never been accepted, this will return -1.

This value should always be greater than startTimestamp and less than endTimestamp
