public function withCustomUserId(customUserId:String):SingularConfig

Send the user ID to Singular. This can be a username, email address, randomly generated string, or whichever identifier you use as a user ID. Singular uses the user ID in user-level data exports as well as internal BI postbacks (if you configure such postbacks).
