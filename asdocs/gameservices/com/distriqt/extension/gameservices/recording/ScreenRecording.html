checkAvailability()method
public function checkAvailability(callback:Function = null):Boolean

Initiates a check to determine whether recording is currently available. It will return the previous known value of the availability.

Note: This may initially return false, but change to true after the availability requirements have been determined. Make sure you use a callback function or listen for the ScreenRecordingEvent.AVAILABILITY_CHANGED event to determine when this value changes.

If a callback function is provided, one Boolean parameter will be passed to the function indicating the availability flag

Parameters

callback:Function (default = null)

Returns
Boolean

Example
Using a callback 
         GameServices.service.recording.checkAvailability( function( available:Boolean ):void
         {
             // available indicates the current value.
         });
Using the changed event 
         GameServices.service.recording.addEventListener( ScreenRecordingEvent.AVAILABILITY_CHANGED, changeHandler );
         GameServices.service.recording.checkAvailability();
                  function changeHandler( event:ScreenRecordingEvent ):void
         {
             GameServices.service.recording.removeEventListener( ScreenRecordingEvent.AVAILABILITY_CHANGED, changeHandler );
             var available:Boolean = GameServices.service.recording.checkAvailability();
         }
