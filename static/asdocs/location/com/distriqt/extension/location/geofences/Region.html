Packagecom.distriqt.extension.location.geofences
Classpublic class Region
InheritanceRegion Object

This class represents a monitoring region. It is a circularly defined area around a geographic point.

You must correctly set the properties of a region to be able to start monitoring the region. This includes the latitude, longitude, radius and identifier values. You can call the isValid function on the region to check your region is valid.

Internally the region is identified by the identifier value, so it is very important that this value be unique.
