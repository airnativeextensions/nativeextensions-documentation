This class contains several helper functions for dealing with records contained in a message. It allows you to easily process the payload on a record and convert it to text or other format as described in the individual functions.

getPayloadAsText()method
public static function getPayloadAsText(record:NdefRecord):String

Attempts to interpret the record as a Text record. This will generally succeed when the tnf is well known and the type of the record is RTD_TEXT. You should wrap these queries in a try/catch statement to make sure you catch any payloads that don't match the text format. 

         try
         {
             var recordText:String = NdefRecordHelper.getPayloadAsText( record );
         }
         catch (e:Error)
         {
             // Not a text record
         }
