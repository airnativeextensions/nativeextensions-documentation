public function setMuteSystemSounds(muteSystemSounds:Boolean):SpeechRecogniserOptions

If true then the extension will attempt to mute any system sounds while listening is in progress. This has mixed success and is a bit of a workaround for the Android start / stop listening sounds, it works in ostcases.

setPreferOnDevice()method 
public function setPreferOnDevice(preferOnDevice:Boolean):SpeechRecogniserOptions

If true, then the extension will attempt to create an on device speech recogniser and request that recognition be attempted on device. This does not guarantee that no network requests will be made.
