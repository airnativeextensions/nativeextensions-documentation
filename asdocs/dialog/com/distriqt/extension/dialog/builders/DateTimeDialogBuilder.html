Classpublic class DateTimeDialogBuilder
InheritanceDateTimeDialogBuilder Object

A Dialog builder helps setup and create a dialog using the Dialog.service.create function.

This dialog builder can be used to create a DateTimeDialogView

setIdentifier()method 
public function setIdentifier(id:int):DateTimeDialogBuilder

Sets a requested value for the identifier (id) of the created dialog. Note that if a dialog of the same id already exists and hasn't been disposed then this value will be ignored and a valid value assigned to the view on creation.

You can check the id through DialogView.id

Parameters

id:int — The requested id to assign to the view

Returns
