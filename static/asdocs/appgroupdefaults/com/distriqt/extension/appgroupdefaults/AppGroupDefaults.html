setup()method 
public function setup(salt:String, appGroupIdentifier:String, type:String):Boolean

Set the extension configuration to correctly get access to the shared defaults. This will set the configuration and initialise the defaults for access. You must call this before attempting to set or retrieve any values. You should wait for the INITIALISED event before attempting to access the data stored in the system.

Parameters

salt:String — A salt value used to encrypt the values in certain circumstances.
 
appGroupIdentifier:String — The "App Group Identifier" configured for the iOS application
 
type:String (default = NaN) — Android only: Either TYPE_CONTENTPROVIDER or TYPE_FILE depending on the method required.

Returns
Boolean — true if successful and false otherwise
