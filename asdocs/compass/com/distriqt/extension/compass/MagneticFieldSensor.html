magnetic:field:uncalibrated:updated Event
Event Object Type: com.distriqt.extension.compass.events.MagneticFieldEvent
MagneticFieldEvent.type property = com.distriqt.extension.compass.events.MagneticFieldEvent.MAGNETIC_FIELD_UNCALIBRATED_UPDATED

Dispatched when the magnetic field changes on the device. This event will contain values in the #fieldX #fieldY #fieldZ parameters specifying the current magnetic values.

The values will be the raw uncalibrated values from the device sensor.

magnetic:field:updated Event  
Event Object Type: com.distriqt.extension.compass.events.MagneticFieldEvent
MagneticFieldEvent.type property = com.distriqt.extension.compass.events.MagneticFieldEvent.MAGNETIC_FIELD_UPDATED

Dispatched when the magnetic field changes on the device. This event will contain values in the #fieldX #fieldY #fieldZ parameters specifying the current magnetic values.

The values will be the calibrated values from the native calibration algorithm.
