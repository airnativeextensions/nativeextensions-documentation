Property Detail
isAvailableproperty
isAvailable:Boolean  [read-only]

true if cloud document synchronisation is available

iCloud Document Sync checks for iCloud availability before performing any iCloud-related operations. Any iCloud Document Sync methods may return prematurely and without warning if iCloud is unavailable. Therefore, you should always check if iCloud is available before performing any iCloud operations.

Note you must call setup to setup your container before this will return true so you should only perform this test after a successful call to setup.

isSupportedproperty 
isSupported:Boolean  [read-only]

Returns true if the current platform (iOS/tvOS/Android etc) supports the functionality defined in this interface.

If this returns false, all other functionality will have undefined results and you should not call any other function.
