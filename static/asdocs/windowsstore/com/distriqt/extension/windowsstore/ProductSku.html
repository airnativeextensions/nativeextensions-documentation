currencyCodeproperty
public var currencyCode:String

A currency code is a three-letter code that is, in most cases, composed of a country’s two-character Internet country code plus an extra character to denote the currency unit. For example, the currency code for the Australian dollar is “AUD”. Currency codes are based on the ISO 4217 standard.

public var priceString:String

The product price represented as a string ready for user display. This will contain the currency sign of the price. Wherever possible you should use this string to display the amount of the product.

sourceproperty 
public var source:String

This will be the original source string of the product matching what the store has sent to the device. Use this if you want to do any specific processing on the product information that you can't perform with the above. Generally you won't need to use this
