Packagecom.distriqt.extension.pdfreader.builders
Classpublic class PDFViewBuilder
InheritancePDFViewBuilder Object

This class is used to build an instance of PDFViewOptions to pass to the PDFReader.service.createView() function.

It simplifies the process of setting the correct options to enable a particular feature of a PDFView

setFitMode()method 
public function setFitMode(mode:int):PDFViewBuilder

Sets the fit mode for scaling the PDF document within the view.

The default FIT_BEST should be used wherever possible. It uses the best possible dimension to fit the document into the view. The other options can be used to force the document to be stretched to the width or height of the view, potentially sacrificing the visibility of the document.

mode:int — A fit mode from PDFViewOptions

public function shouldOverrideLinks(enabled:Boolean):PDFViewBuilder

Sets whether links in the PDF to websites should be automatically opened.

The default, false: i.e. links should not be overriden, causes any links clicked in the PDF to be opened in the native browser.

If you do override links then you should handle the link operation in the PDFViewLinkEvent.CLICK event for the PDF View.

Examples
The simplest usage of the builder 
     var path:String = File.applicationDirectory.resolvePath( "document.pdf" ).nativePath;
     var view:PDFView = PDFReader.service.createView( 
                         new PDFViewBuilder()
                             .setPath( path )
                             .build()
                         );
     view.show();
The following example shows the usage of the builder to control some of the view options 
     var path:String = File.applicationDirectory.resolvePath( "document.pdf" ).nativePath;
     var view:PDFView = PDFReader.service.createView( 
                         new PDFViewBuilder()
                             .setPath( path )
                             .setPassword( "password" )
                             .showThumbnails( true )
                             .showEmail( true )
                             .setEmailContent( "Test Subject", "Some simple content" )
                             .build()
                         );
     view.show();
