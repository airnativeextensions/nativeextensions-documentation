Classpublic class DialogParameters
InheritanceDialogParameters Object

This class defines the parameters used to create a dialog view. Generally you should not use this class, but instead use the builders available which will generate this class based on a specific type of dialog.

cancelableproperty 
cancelable:Boolean

If true then the dialog can be cancelled by the user clicking the background or outside the dialog. This allows a user to cancel an operation.

Alternatively if this is false then user interaction can not stop the indictor and you should use dismiss when you need to remove the dialog

This is only applicable to certain dialogs, see the individual builders for more information.

disposeOnCloseproperty 
disposeOnClose:Boolean

If true this dialog is destroyed as soon as it is dismissed.

If false the dialog remains initialised allowing quick display of the dialog at a later time. You must call dispose on the DialogView to remove the dialog from memory completely.
