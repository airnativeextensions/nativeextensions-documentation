Interfacepublic interface BrowserView extends flash.events.IEventDispatcher

The BrowserView simplifies the process of having to display content from external urls. The creates a browser like experience with having your users leave your application.

While you can do most of this through the WebView, the BrowserView is the suggested method of displaying external content on both Android and iOS.

  • On iOS this uses the Safari View Controller
  • On Android this uses Chrome Custom Tabs

prepare()method 
public function prepare():Boolean

This call prepares the browser view for loading content. You must call this function and wait for a READY event before calling openWithUrl.

If you don't call this function the view should still display you just might not receive events about the view.

Returns
Boolean — true if the prepare operation started successfully
