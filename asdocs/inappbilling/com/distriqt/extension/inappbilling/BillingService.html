Property Detail
googlePlayPublicKeyproperty
public var googlePlayPublicKey:String

The Google Play Public Key is your Google Play License Key for the application. It is used to verify the signature of purchases to prevent fraudulent purchases. See the main documentation for more information on acquiring this key from the Google Play Developer Console. Please note it's highly recommended that you attempt to obfuscate this key in your code. See the security recommendations in the main documentation.

huaweiAppGalleryPublicKeyproperty 
public var huaweiAppGalleryPublicKey:String

The Huawei AppGallery Public license key. Similar to the googlePlayPublicKey this is used to verify the signature of purchases to prevent fraudulent purchases.

operationModeproperty 
public var operationMode:String = production

Certain services allow setting an operation mode, such as a sandbox or test environment mode. These modes change the operation of the library and are generally only used for development. Unless directed otherwise, this should always be set to OPERATION_MODE_PRODUCTION for any builds uploaded to a store.
