This class represents the NativeText extension.

This extension allows you to create a native text field that can be placed above the AIR content on the stage. You cannot place other flash content above these fields.

The extension gives you the ability to finer control of the type of controls the user has access to while entering text into the text field, by being able to disable the normal action menus and disabling auto suggestion/correction functionality.

You create a native text field by calling the createTextField function, passing in any options required for the text field. This will return a reference to the text field that can be used to listen for events and manipulate the native text field.

The functionality and initial display of the text field is controlled by the settings in the NativeTextFieldOptions class. In this class you can control the way the text field operates, such as multi-line, text and background colours, font size, width/height and disabling functionality. See this class for more information on the features.

When you create a text field it is added to the display immediately. And when you destroy it, it is removed.
