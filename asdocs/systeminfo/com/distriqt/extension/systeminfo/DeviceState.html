isDeviceIdleModeproperty 
isDeviceIdleMode:Boolean  [read-only]

Returns true if the device is currently in idle mode. This happens when a device has been sitting unused and unmoving for a sufficiently long period of time, so that it decides to go into a lower power-use state. This may involve things like turning off network access to apps.

You can monitor for changes to this state with `DeviceStateEvent.IDLE_MODE_CHANGED`.

Supported on Android devices with API v23 or higher. Will return false on unsupported devices.


Implementation

isPowerSaveModeproperty 
isPowerSaveMode:Boolean  [read-only]

Returns true if the device is currently in power save mode. When in this mode, applications should reduce their functionality in order to conserve battery as much as possible.

You can monitor for changes to this state with `DeviceStateEvent.POWER_SAVE_MODE_CHANGED`.
