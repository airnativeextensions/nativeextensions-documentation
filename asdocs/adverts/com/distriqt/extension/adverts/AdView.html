setAdaptiveAdSize()method 
public function setAdaptiveAdSize(width:int = -1, orientation:String = auto):*

Sets an adaptive advert size based on the requested width and orientation. If left to the defaults the extension will apply the current available width and device orientation to calculate these values. Note: You must set the ad size before loading.

Parameters

width:int (default = -1) — The available width for the adaptive advert
 
orientation:String (default = auto) — The requested orientation, or "auto" for the current orientation

Returns
