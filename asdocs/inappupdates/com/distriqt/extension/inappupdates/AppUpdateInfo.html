Property Detail
availableVersionCodeproperty
public var availableVersionCode:int = -1

If an update is available or in progress, this method returns the version code of the update. If no updates are available, this method returns an arbitrary value.

bytesDownloadedproperty 
public var bytesDownloaded:Number = -1

Returns the number of bytes downloaded so far. The return value is defined only if installStatus() returns InstallStatus.DOWNLOADING.

clientVersionStalenessDaysproperty 
public var clientVersionStalenessDays:int = -1

If an update is available or in progress, this method returns the number of days since the Google Play Store app on the user's device has learnt about an available update. If update is not available, or if staleness information is unavailable, this returns -1.

installStatusproperty 
public var installStatus:int = 0

Returns the progress status of the update. The return value is defined only if updateAvailability returns UpdateAvailability.DEVELOPER_TRIGGERED_UPDATE_IN_PROGRESS. To see examples, go to Install a flexible update.

See also

InstallStatus.CANCELED

totalBytesToDownloadproperty 
public var totalBytesToDownload:Number = -1

Returns the total number of bytes to be downloaded for this update. The return value is defined only if installStatus() returns InstallStatus.DOWNLOADING.

updateAvailabilityproperty 
public var updateAvailability:int = 0

Returns a value from UpdateAvailability indicating whether an update is available for the app.

UpdateAvailability.UNKNOWN

updatePriorityproperty 
public var updatePriority:int = -1

Returns the in-app update priority for this update, as defined by the developer in the Google Play Developer API. You can define in-app update priority of a release by setting 'inAppUpdatePriority' field under Edits.tracks.releases. The return value is defined only if updateAvailability() returns UpdateAvailability.UPDATE_AVAILABLE.
