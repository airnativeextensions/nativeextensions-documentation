Property Detail
attemptSilentSignInproperty
public var attemptSilentSignIn:Boolean = true

Set whether to attempt a silent sign in on startup if the user has previously signed in. This will only work if the user has previously signed in. If true the setup process and the user has previously signed in then there will be a UI presented to the user showing the signing in process and the setup process with be asynchronous. If false then the setup process will be synchronous and will not present any UI.

The default value is true.
